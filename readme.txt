Notes:
* Lobby stuff is done in the console for client

Controls:
* Move Up: 'W' Key
* Move Down: 'S' Key
* Move Left: 'A' Key
* Move Right: 'D' Key
* Toggle Console: '~' Key

Console Commands:
* "quit": quits program
* "clear": clears console
* "changeIP <string ipAddr>": changes IP address that client connects to
* "changePort <int portNum>": changes port number that client connects to
* "showGames": get a list of games available in the lobby
* "createGame": create a new game in the lobby and start playing
* "joinGame <int gameID>": join existing game with given game ID (can be seen through showGames)
